HK-Aufnahme App - Kurzanleitung
================================

1. APP STARTEN
--------------
Die App im Browser unter folgendem Link öffnen:

    https://e1felixr.github.io/heizkoerper/

Keine Installation nötig - die App läuft direkt im Browser.

WICHTIG: Bitte Chrome oder Edge verwenden.
Andere Browser (Samsung Internet, Firefox) können zu abweichender
Darstellung bei Auswahlfeldern führen.


2. AUF DEM STARTBILDSCHIRM SPEICHERN (empfohlen)
-------------------------------------------------
Die App lässt sich wie eine normale App auf dem Startbildschirm ablegen.
Danach startet sie im Vollbildmodus und funktioniert auch offline.

Android (Chrome):
  - Link im Chrome-Browser öffnen
  - Auf die drei Punkte (Menü) oben rechts tippen
  - "Zum Startbildschirm hinzufügen" oder "App installieren" wählen
  - Bestätigen

Android (Edge):
  - Link im Edge-Browser öffnen
  - Auf die drei Punkte (Menü) unten tippen
  - "Zum Startbildschirm hinzufügen" wählen
  - Bestätigen

iPhone / iPad (Safari):
  - Link in Safari öffnen
  - Auf das Teilen-Symbol (Quadrat mit Pfeil) tippen
  - "Zum Home-Bildschirm" wählen
  - "Hinzufügen" tippen


3. ERSTE SCHRITTE
-----------------
  - Beim ersten Start wird nach dem Erfasser-Namen gefragt (Pflichtfeld)
  - Ein neues Projekt anlegen (z.B. "Musterstraße 12")
  - Heizkörper mit "+" erfassen
  - Fotos können direkt über die Kamera aufgenommen werden


4. DATEN VERSENDEN
------------------
  - In der Heizkörper-Liste auf "Versenden" tippen
  - Empfänger auswählen und absenden
  - Die Daten werden als ZIP-Datei (Excel-Tabelle + Fotos) versendet


5. OFFLINE-NUTZUNG
------------------
Die App funktioniert vollständig offline. Alle Daten werden lokal auf
dem Gerät gespeichert. Eine Internetverbindung wird nur benötigt für:
  - Erstmaliges Laden der App
  - Updates der App
